# Tijorat & Rushd CRM

Внутренняя CRM-система компании. Стек: **React + Vite + TypeScript + Tailwind + Supabase**.

## Запуск (локально)

```bash
npm install
cp .env.example .env   # значения Supabase заполняются на Фазе 1
npm run dev
```

Открой адрес из консоли Vite (обычно `http://localhost:5173`).
