# Таджикские строки для проверки (передать GPT)

## Инструкция для GPT

Ты — редактор таджикского языка (тоҷикӣ, кириллица с буквами ғ ӣ қ ӯ ҳ ҷ).
Ниже строки интерфейса CRM в формате:  `ID | RU: <русский> | TG: <текущий таджикский>`.
Проверь и при необходимости исправь ТОЛЬКО таджикский текст (TG) — сделай его
грамотным, естественным и деловым. Русский (RU) — это смысл-оригинал, его не меняй.

ВЕРНИ РЕЗУЛЬТАТ строго построчно в формате `ID = исправленный таджикский`
(один ID на строку, без пояснений, без русского). Сохрани все ID как есть.
Термины держи единообразно во всём тексте.
